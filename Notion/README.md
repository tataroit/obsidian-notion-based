## Точки входа

### Для владельца

| Файл                             | Для чего                                                                         |
| -------------------------------- | -------------------------------------------------------------------------------- |
| `Home.md`                        | Открывай каждую неделю. Статус по всем направлениям, задачи, еженедельный ритуал |
| `База знаний/04. Жизнь/Жизнь.md` | Мастер-документ. Цели, ценности, вектор. Сверяй курс когда теряешь ориентир      |

### Для ИИ

| Файл | Для чего |
| ---- | -------- |
| `_meta/README.md` | Читать первым. Контекст о владельце, структура базы, правила работы с контентом |
| `_meta/Планер задач.md` | Список задач для реализации в текущей или следующей сессии |

## Сценарии использования

**Раз в неделю** → `Home.md` → пройти чеклист → заполнить рефлексию

**Пришла мысль** → блок "Входящие" в `Home.md`

**Новая тема для изучения** → открыть сессию с Claude, сослаться на `_meta/README.md`
