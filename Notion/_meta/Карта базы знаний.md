# Карта базы знаний
_Последнее обновление: 2026-02-24_

---

## Куда смотреть первым делом

| Файл | Для чего |
|---|---|
| `Home.md` | Открываешь каждую неделю. Статус по всем направлениям, активные задачи, еженедельный ритуал |
| `Жизнь.md` | Мастер-документ. Цели, ценности, вектор. Сверяешь курс когда теряешь ориентир |
| `Inbox.md` | Быстро скидываешь любую мысль. Разбираешь раз в неделю |

---

## Домены

| Папка | Что внутри |
|---|---|
| `Карьера/Карьера.md` | Обзор: статус, задачи, мысли по карьере |
| `Карьера/knowledge/` | База знаний по разработке: Dev.md, Development.md |
| `Финансы/Финансы.md` | Обзор: статус, задачи, мысли по трейдингу |
| `Финансы/knowledge/` | База знаний по крипте: скальпинг, индикаторы, торговля |

---

## Служебные папки

| Папка | Что внутри |
|---|---|
| `Заметки/` | Развёрнутые размышления. Сюда — когда думаешь в процессе записи. Сюда же — еженедельные рефлексии с датой |
| `_meta/` | Инструкции для ИИ, шаблоны, эта карта |
| `_temp/` | Пустышки и устаревшее. Не трогать, не читать |
| `Архив/` | Завершённое и неактуальное |

---

## Как работать

**Раз в неделю:** открыть `Home.md` → пройти по чеклисту → скопировать шаблон рефлексии в `Заметки/`

**Новая тема для изучения:** взять шаблон из `_meta/` → дать задание Claude → положить результат в нужный `knowledge/`

**Пришла мысль:** → `Inbox.md`

**Хочу подумать развёрнуто:** → новый файл в `Заметки/`
