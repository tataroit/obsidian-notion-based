# Планер задач для ИИ

Список задач для реализации в будущих сессиях.
Разбираем по одной, последовательно.

---

## В работе / следующие

- [ ] **Регулярные операции по расписанию** — создать блок команд, которые запускаются с заданной периодичностью:
  - Валидация правового аспекта по финансам (актуальность законодательства, налоги, ограничения для РФ)
  - Мониторинг санкционных угроз и реагирование (что изменилось, что нужно пересмотреть)
  - Обновление базы знаний по разработке (что нового вышло во фреймворках, инструментах)
  - Общая рефлексия над базой знаний: что актуально, что устарело, что нужно добавить или пересмотреть

---

## Идеи на потом

- [ ] **Операция "Взгляд со стороны"** — ИИ самостоятельно анализирует базу знаний, рефлексии и паттерны, и ищет неочевидное: что упускается, где есть слепые пятна, какие связи между доменами не замечаются, что стоит пересмотреть. Выдаёт 3-5 инсайтов которые трудно увидеть изнутри. Запускается по запросу, не автоматически.

- [ ] **Детализировать домен Карьера** — по аналогии с Финансами разбить на суб-направления (например: текущая работа / тимлид, профессиональный рост, возможная независимость или бизнес). Понять, есть ли что раскладывать.

- [ ] **Детализировать домен Здоровье** — возможные суб-направления: тело и тренировки, питание, ментальное здоровье. Разложить по аналогии с Финансами.

- [ ] **Структурировать домен Жизнь** — сейчас один файл без содержания. Понять, что туда вообще входит: отношения, досуг, смыслы, окружение — и придать структуру.

- [ ] **Добавить верхний уровень целей** — документ "Где я хочу быть через 3 года" или аналог. Приоритеты между доменами: что важнее, если время ограничено. Сейчас каждый домен живёт сам по себе.

- [ ] **Трекинг прогресса во времени** — что-то, что накапливает итоги: "что изменилось за квартал", "как менялись статусы доменов". Сейчас рефлексии есть, но нет агрегации за длинный период.
