## Входящие
_Новые задачи — скидывай сюда. Разбирай при открытии базы._
### Чеклист запуска крипто-инвестиций
- [ ] 1. Уточнить в Точке о возможности следующих шагов
- [ ] 2. Найти ОКВЭД 64.99, запросить добавление через бухгалтера Точки
- [ ] 3. Подключить ОКВЭД к ИП (~5 рабочих дней)
- [ ] 4. Настроить раздельный учёт в КУДиР: патентные доходы отдельно, крипто-доходы по УСН 6%
- [ ] 5. Предупредить банк об операциях с криптой
- [ ] 6. Зарегистрироваться на Цифра Маркет, разобраться физик/юрик, какие документы выдают
- [ ] 7. Купить аппаратный кошелёк (Ledger/Trezor), настроить
- [ ] 8. Настроить Bybit: KYC, 2FA, whitelist адресов вывода
- [ ] 9. Пересмотреть индикаторы Пифагора, определить точки входа
- [ ] 10. Определить стратегию входа: сразу / частями / на откатах
- [ ] 11. Провести тестовый круг: рубли → Цифра Маркет → USDT → Bybit спот → вывод обратно
- [ ] 12. Завести основные суммы
- [ ] 13. Настроить систему хранения документов по каждой операции


- [ ] Перевыпустить электронную подпись
- [ ] По крипте - разобраться с тем, как купить
- [ ] Создать акк или лучше перейти под рефералку к пифагору и поулчить его индикаторы. И начать  работать с ними
- [ ] Общая стратегия - покупки на лоях, продажи по частям на отскоках и тянуть выше
- [ ] Пересмотреть его последние видосы по индикаторам и точкам входа
- [ ] Поставить постоянную операция - по проверки состояния на рынке по его индикаторам и понимания реагировать или нет
- [ ] Собрать виджет в ЕМ, дашборд с общим выводом о состоянии рынка
- [ ] Купить какой-то симулятор мышки адекватный
- [ ] Поискать альтернативы - стоит ли разрабатывать или покупать что-то другое, что может более точно и адекватно симулировать устройство?


- [ ] Как инвестироват ьв антропик? операцию на отслеживание новостей и возможных действий связанных с этим
